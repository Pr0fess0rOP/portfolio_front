# Pathik Viramgama Portfolio Website

A static portfolio website built from the resume information available in the file library and the provided LinkedIn URL.

## Files
- `index.html` — main website
- `style.css` — responsive styling
- `script.js` — mobile menu + footer year

## How to use
Open `index.html` in any browser. For deployment, upload the folder to GitHub Pages, Netlify, Vercel, or any static hosting provider.
